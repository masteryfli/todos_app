# todo-list
这是一个用vue3 + ts 实现的 todoList 的app,中间涉及了vue3 中常用的api 和ts 语法,数据存储在localstorage中 

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
